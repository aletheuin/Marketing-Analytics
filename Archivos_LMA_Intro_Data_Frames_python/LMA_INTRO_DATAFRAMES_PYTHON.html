Creación de un DataFrame¶

1.- A través de Series¶

Tenemos que tener en cuenta que las series funcionan de forma similar a un diccionario de datos donde tenemos los archivos formados por {clave_1: 'valor uno', clave_2. 'valor dos', ....., clave_n: 'valor m'}. Las series pueden formarse a travs de un numpy array o una lista y unos valores que actuarn como ndices (si no se introducen se tomarn por defecto de 0 a la longitud del conjunto de elementos).

In [2]:
# Ejemplo 1 a travs de Series
series_lista = pd.Series([46,41,36,33], index=['FC BARCELONA', 
                                   'ATLETICO', 
                                   'REAL MADRID',
                                   'SEVILLA'] )

series_lista

ATLETICO        41

Obtener multiples filas de un dataframe¶

Por lo general, podemos utilizar las mismas propiedades que las de las listas, salvo seleccionar una nica fila que es diferente:

  • Para seleccionar desde la primera fila hasta un lmite realizamos: __dataframe[0:n]__, anlogamente podemos omitir el cero y realizar simplemente: __dataframe[ :n]__
  • Para seleccionar desde una fila hasta el final realizamos: __dataframe[n:]__
  • Para seleccionar un rango definido de filas realizamos: __dataframe[n:m]__

Resumen función iloc¶

Con la funcin .iloc podemos seleccionar varias filas de una forma muy sencilla que puede resumirse como:

  • dataframe.iloc[0] - Primera fila de un dataframe.
  • dataframe.iloc[1] - Segunda fila de un dataframe.
  • dataframe.iloc[-1] - ltima fila de un dataframe
  • dataframe.iloc[[n]] - Fila n del dataframe.
  • dataframe.iloc[n:m] - Rango personalizado de las filas n a m

En columnas el resumen de la funcin .iloc pasara a ser:

  • dataframe.iloc[:, 0] - Primera columna de un dataframe.
  • dataframe.iloc[:, 1] - Segunda columna de un dataframe.
  • dataframe.iloc[-1] - ltima columna de un dataframe
  • dataframe.iloc[:,[n,m]] - Exactamente, las columnas n y m de un dataframe.
  • dataframe.iloc[:, n:m] - Rango personalizado de las columnas n a m de un dataframe.

Podemos tambin realizar una seleccin mltiple de columnas filas con .iloc con los siguientes ejemplos:

  • dataframe.iloc[[0,5,7,9], [1,4]] - Seleccin de las filas 0,5,7,9 de las columnas 1 y 4
  • dataframe.iloc[0:10, 0:2] - Seleccin basada en rangos de las filas 0 a 10 de las columnas 0 a 2

Algunas funciones básicas de un dataframe¶

A continuacin vamos a ver un listado de algunas de las principales funciones que podemos aplicar en un dataframe para operar en sus columnas:

  • Obtener las primeras filas de un dataframe con .head
  • Obtener las ltimas filas de un dataframe con .tail
  • Obtener los elmentos nicos de una columna de un dataframe con .unique
  • Obtener un resumen estadstico del dataset con .describe
  • Obtener la media de una columna con .mean
  • Obtener una copia de un dataframe con .copy
  • Ver los nombres de las columnas de un dataframe con .columns
  • Obtener la correlacin entre todas las variables numricas con .corr
  • Borrar duplicados de un dataframe con .drop_duplicates
  • Especialmente para Big Data sets podemos ver el consumo en memoria RAM de nuestro dataset con .memory_usage
  • Ver un resumen resumen grfico (basado en densidad, scatter plots y grficas de correlaciones) de todas las variables del dataframe con .scatter_matrix
  • Obtener un histograma de cada variable numrica del dataset con .hist
